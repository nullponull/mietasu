# Basic Operations

This spec describes the basic navigation and interaction flows for the Mietasu LP and Demo environment.

## Scenario: Navigate to Demo Dashboard

1.  **Start** at the Home page (`/`).
2.  **Click** the "デモ画面を見る" (View Demo) button in the header or hero section.
3.  **Verify** that the URL changes to `#/demo`.
4.  **Verify** that the Dashboard page is displayed with the title "ダッシュボード".
5.  **Verify** that the sidebar contains links for "ナレッジベース", "議事録・録音", "AIチャット", and "設定・ユーザー".

## Scenario: Check Knowledge Base

1.  **Navigate** to the Demo Dashboard.
2.  **Click** on "ナレッジベース" in the sidebar.
3.  **Verify** that the Knowledge Base page is displayed.
4.  **Verify** that there is a list of knowledge items (e.g., "技術開発部", "営業企画部").

## Scenario: Use AI Chat

1.  **Navigate** to the Demo Dashboard.
2.  **Click** on "AIチャット" in the sidebar.
3.  **Verify** that the Chat interface is displayed.
4.  **Click** on one of the preset questions (e.g., "最新の議事録を要約して").
5.  **Verify** that a response is generated and displayed in the chat window.
