# Test Plan for Mietasu LP Mockup

## 1. Dashboard
- [ ] **Notification Center**: Click the bell icon in the header. Verify that the dropdown opens and displays notifications. Check if unread indicators are visible.
- [ ] **Sticky Notes**: Verify that sticky notes are displayed with different colors and rotations. Try adding a new note. Check if the "Chat Quote" note is displayed correctly. Hover over notes to see the delete button.
- [ ] **Project List Preview**: Verify that recent projects are listed and clickable.

## 2. Project Management
- [ ] **Project List**: Navigate to `/projects`. Verify the list of projects, status badges, and "New Project" button.
- [ ] **Project Detail**: Click on a project. Verify the project dashboard, member list, and recent files.

## 3. Meeting Minutes & Files
- [ ] **Meeting Detail**: Navigate to a meeting detail page. Verify the AI summary, transcript, and audio player placeholder.
- [ ] **File Detail**: Navigate to a file detail page. Verify the new file preview (mock PDF), version history, and comments section.

## 4. Chat
- [ ] **Chat List**: Navigate to `/chat`. Verify the list of chat threads.
- [ ] **Chat Detail**: Click on a chat thread. Verify the chat interface, bot response with citations, and input field.

## 5. Search
- [ ] **Search Bar**: Use the search bar in the header. Enter a keyword (e.g., "API") and press Enter.
- [ ] **Search Results**: Verify the search results page with tabs (All, Project, File, Minutes, Chat) and highlighted keywords.

## 6. Settings
- [ ] **Individual Settings**: Navigate to `/settings`. Verify the profile, notification, and appearance settings.
- [ ] **Admin Dashboard**: Navigate to `/admin`. Verify the tabs for User, Role, and Project management.
